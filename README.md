# Expense Manager Dashboard

A premium financial dashboard designed to manage and export company expenses.

## Features
- **Grouped Views**: See exactly what's happening in each company with dedicated cards and automatic totals.
- **Monthly History**: Navigate through past months using the sidebar. Data is automatically grouped by the date you input.
- **Smart Data Persistence**: Your expenses, custom companies, and descriptions are saved locally in your browser.
- **Excel Export**: Download a professional Excel report for the selected month, formatted with companies in separate columns.
- **Customizable**: Add or remove companies and common descriptions via the "Categories" settings.

## How to use
1. Open `index.html` in any modern web browser (Chrome, Safari, Edge).
2. Click **"Add Expense"** to record a new transaction.
3. Use the **Sidebar** to switch between different months.
4. Click **"Export Excel"** to download the report for the current month.
